# CLAUDE.md — dimasma0305.github.io

Personal portfolio + blog (Dimas Maulana). Next.js App Router, static-exported to
GitHub Pages. Canonical domain is **dimasc.tf**; `dimasma0305.github.io` 301-redirects there.

## Stack
- Next.js 15.2.4 (App Router, `output: 'export'`, `trailingSlash: true`)
- React 19 · TypeScript 5 · Tailwind 3.4 · shadcn/ui (Radix UI)
- Package manager: **bun** (`bun.lock`)
- Content: blog + notes sourced from **Notion** at build time; RSS/indexes generated by `scripts/`

## Commands
- Dev server:       `bun run dev`
- Build (→ `out/`): `bun run build`        # static export
- Lint:             `bun run lint`
- Regenerate content: `bun run refresh-content`   # Notion → posts + index + RSS; needs NOTION_TOKEN, NOTION_DATABASE_ID
- No test suite configured.

## Notes
- **Static export**: no server runtime or API routes. Keep links/canonicals consistent with `trailingSlash: true`.
- **Brand mark**: `components/logo-mark.tsx` (an open ring + two-tone "dm", outlined Inter paths — do not replace with `<text>`). Icons live in `public/`; `public/logo.png` is referenced by the JSON-LD Organization/Person logo, so keep it a real file or search engines 404 it.
- **XSS surface**: blog/notes HTML comes from Notion and is rendered via `dangerouslySetInnerHTML`. Escape/sanitize at every sink — see `safeJsonLd()` in `components/seo.tsx` and `sanitizeUrl()` in `lib/notion-content-utils.ts`.
- **Services**: `/services` is a dedicated route (`app/services/page.tsx` + `components/services-detail.tsx`); the homepage shows a teaser (`components/services-section.tsx`).

## Conventions
- Match the existing code: dark-only theme, a single blue accent, semantic design tokens. Prefer no em dashes in user-facing copy.
- Global ECC rules/skills are already installed under `~/.claude/` (no project-local `.claude/` needed).
